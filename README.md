# KASA — Personal Finance Terminal

Brutalistický finance tracker s terminálovou estetikou. Žiadne frameworky, žiadne závislosti — čistý HTML, CSS a JavaScript.

## Funkcie

- Animovaný loading screen s progress barom
- Sledovanie príjmov a výdavkov
- 8 kategórií transakcií
- Filtrovanie (všetky / príjmy / výdavky)
- Balance progress bar
- Všetko sa ukladá v prehliadači (localStorage)
- Responzívny dizajn

## Štruktúra

```
finance-tracker/
├── index.html   — štruktúra
├── style.css    — dizajn & animácie
├── app.js       — logika
└── README.md
```

